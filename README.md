# 여러 전시회의 정보를 한눈에 볼 수 있는 전시회 통합 웹사이트(프론트엔드)

## 📀 프로젝트 내용

- 월별 전시 페이지
    - 월별로 진행되는 전시회의 정보를 나열하고 예매 사이트로 연결해준다
- 인기 전시 페이지
    - 인기 있는 전시의 정보와 위치정보를 보여준다
- 온라인 전시 페이지
    - 사이트에서 작품을 보여주고 마우스를 올리면 작품명, 작가, 작품 설명을 보여준다

## 📀 개발 환경

### 프론트엔드

- HTML
- CSS
